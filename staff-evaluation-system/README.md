# TRY WASH スタッフ評価システム

代表者・オーナー専用の独立システムです。

予約管理システムとは画面・保存キーを分離しています。現段階では静的サイト上の簡易アクセスゲートです。本番運用ではサーバー側ログイン、ユーザー権限、監査ログ、暗号化されたデータベース保存に置き換えてください。

## 閲覧可能ユーザー

- 代表者
- オーナー

一般スタッフ、管理者権限スタッフ、店舗責任者は閲覧不可です。

## 管理項目

- 基本情報: 氏名、入社日、役職、雇用形態
- 売上関連: 担当件数、担当売上、平均単価、利益額
- レバレート: 作業時間、作業人数、平均レバレート、最高レバレート、月間レバレート
- 評価: 技術力、接客力、提案力、SNS貢献、店舗貢献、総合評価
- 面談記録: 評価コメント、面談内容、課題、改善内容、目標設定

## レバレート定義

```text
レバレート = 売上 ÷ 作業人数 ÷ 作業時間
```

## 簡易アクセス

静的版の初期パスコードは `staff-evaluation-system/app.js` の `ownerPasscode` で変更できます。
